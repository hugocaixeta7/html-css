Resumo de CSS

1 Fonte padrão no body

                body {
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 16px;
                    font-weight: normal;
                    font-style: italic;
                }

2 Medidas de Fonte

  • Absolutas: cm, mm, in, px, pt, pc
  • Relativas: em, rem, %, vw, vh, ex

16px geralmente = 1em

3 Estilos de Texto

                p {
                    font-style: italic; /* normal | italic | oblique */
                    font-weight: bold;  /* lighter | normal | bold | bolder | 100-900 */
                }

4Shorthand (atalho)

                h1 {
                    font: italic bold 2em 'Times New Roman', Times, serif;
                }

Ordem: font-style → font-weight → font-size → font-family

5 Decoração de Texto

h2 {
                text-decoration: underline; /* underline | overline | line-through | none */
            }

✨ Resumo Visual

  • @charset "UTF-8"; Define a codificação de caracteres para o arquivo CSS.
  • font-family: Define a fonte principal.
  • font-size: Controla o tamanho do texto.
  • font-weight: Define o peso do texto.
  • font-style: Define se é itálico onormal.
  • text-decoration: Adiciona efeitovisuais ao texto.
  • background-image: Define uma imagem de fundo para um elemento.
  • background-attachment: Determina se a imagem de fundo será fixa ou rolará junto com a página.
  • background: Propriedade abreviada para definir cor de fundo, imagem, posição etc.
  • background-color: Define a cor de fundo do elemento.
  • color: Define a cor do texto dentro de um elemento.
  • border-radius: Define o raio das bordas do elemento, permitindo cantos arredondados.
  • border-bottom: Define propriedades da borda inferior do elemento.
  • padding-bottom: Define o espaço interno (preenchimento) na parte inferior do elemento.
  • padding-left: Define o preenchimento do lado esquerdo do elemento.
  • overflow-x: Especifica se o conteúdo deve ser cortado.
  • margin: Define o espaço externo ao redor do elemento.
  • width: Define a largura do elemento.
  • padding: Define o preenchimento interno do elemento.
  • box-shadow: Aplica sombras ao redor do contorno do elemento.
  • line-height: Define a altura das linhas de texto.
  • text-align: Especifica o alinhamento horizontal do texto dentro do elemento.
  • text-shadow: Adiciona sombra ao texto.
  • text-indent: Especifica o recuo da primeira linha de texto dentro do elemento.
  • font: Propriedade abreviada para definir estilo, variante, peso, tamanho da fonte, altura dalinha e família da fonte.
  • id: Define um identificador único para um elemento.
  • class: Define uma classe para um elemento, permitindo a aplicação de estilos a vários elementoscom a mesma classe.
